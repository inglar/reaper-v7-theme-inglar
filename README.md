# REAPER V7 Inglar Theme
[![latest release badge](https://img.shields.io/github/v/release/inglar/reaper-v7-theme-inglar?sort=semver)](https://github.com/inglar/reaper-v7-theme-inglar/releases/latest)
[![license badge](https://img.shields.io/github/license/inglar/reaper-v7-theme-inglar)](./LICENSE)

Modified original theme to add readability, contrast and brightness. Especially for Macbook Retina screens. **Works only for 200 layout**.

## Install
1. Download latest from [Releases](https://github.com/inglar/reaper-v7-theme-inglar/releases/latest)
2. Copy to `~/Library/Application Support/REAPER/ColorThemes`
